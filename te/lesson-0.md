# ప్రోగ్రామింగ్ (కోడింగ్) అంటే ఏమిటి?

గణన పనిని నిర్వహించడానికి కంప్యూటర్‌కు ఇచ్చిన సూచనల సమితి.

## కమ్యూనికేషన్

సాధారణ ఆదేశంతో కంప్యూటర్‌తో మాట్లాడండి

```py
print('hi')
```

## సూచనల సమితి

కంప్యూటర్ సూచనల సమితిని అమలు చేయగలదు

```py
print('hi')
a = 10
print(a)
```

## పునరావృతం

```
def add(a, b):
	print(a + b)

add(5, 10)
add(2, 3)
add(15, 10)
add(20, 30)
```
